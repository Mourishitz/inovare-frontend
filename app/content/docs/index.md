# Bem-vindo ao Inovare

Inovare é um sistema completo para gerenciamento de chás de lingerie.

## Para Noivas

Como noiva anfitriã, você pode:

1. **Criar seu Chá**: Cadastre os detalhes do seu evento
2. **Definir Preferências**: Escolha tema, orçamento e número de convidadas
3. **Revisar Catálogo**: Visualize e aprove as peças íntimas selecionadas
4. **Acompanhar Status**: Veja o progresso da organização

## Para Administradores

Como administrador, você tem acesso a:

1. **Dashboard Completo**: Visão geral de todos os chás
2. **Gerenciamento de Chás**: Acompanhe cada evento
3. **Criação de Catálogos**: Monte catálogos personalizados de lingerie
4. **Gestão de Produtos**: Mantenha o catálogo de peças íntimas atualizado
